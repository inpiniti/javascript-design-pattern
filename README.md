## 시작하기

이 프로젝트를 시작하려면 저장소를 복제하고 프로젝트 디렉토리로 이동하세요. `src` 폴더에서 각 디자인 패턴의 구현을 탐색할 수 있습니다. 각 패턴에는 고유한 JavaScript 파일과 해당 목적과 사용법을 설명하는 마크다운 문서가 있습니다.
